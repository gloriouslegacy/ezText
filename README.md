# ezText

텍스트 단축키 매크로 프로그램 - Windows용 텍스트 입력 자동화 도구

## 기능

- ⌨️ 사용자 정의 단축키로 텍스트 자동 입력
- 🌓 다크/라이트 테마 
- 🌐 다국어 지원 (한국어, English)
- 🔄 자동 업데이트
- 💾 단축키 저장/불러오기
- 📋 시스템 트레이 아이콘 지원

## 다운로드

[Releases 페이지](https://github.com/gloriouslegacy/ezText/releases)에서 최신 버전을 다운로드하세요.

### 설치 방법

1. `ezText-setup.exe` 다운로드
2. 다운로드한 파일 실행
3. 설치 마법사 안내에 따라 설치 진행
   - 설치 경로: `%APPDATA%\ezText`

## 사용 방법

1. **단축키 추가**
   - 텍스트 입력란에 입력할 텍스트 입력
   - 단축키 입력란 클릭 후 원하는 키 조합 입력
   - "추가" 버튼 클릭

2. **단축키 사용**
   - 등록한 단축키를 눌러 텍스트 자동 입력

3. **단축키 관리**
   - 테이블에서 텍스트/단축키 더블클릭하여 수정
   - 체크박스 선택 후 "선택 삭제" 버튼으로 삭제
   - "전체 삭제"로 모든 단축키 제거

4. **파일 관리**
   - `파일 > 저장`: 현재 단축키 저장
   - `파일 > 다른 이름으로 저장`: 새 파일로 저장
   - `파일 > 불러오기`: 저장된 단축키 불러오기

5. **설정 관리**
   - `설정 > 시작프로그램 등록`: 윈도우 로그인시 자동 실행

## 자동 업데이트

- 프로그램 시작 시 자동으로 업데이트 확인
- 새 버전 발견 시 트레이 알림 표시
- `도움말 > 업데이트 확인` 메뉴에서 수동 확인 가능
